🔧 How It Works

  • Automatic Detection: The extension detects browser language (Turkish/English) on first use
  • Manual Selection: Users can change language via dropdown in popup or settings
  • Persistent Storage: Language preference is saved in Chrome sync storage
  • Dynamic Translation: All UI text is loaded from locale files at runtime
  • Jira Issues: Even the Jira issue content (ADF format) is translated

🎯 Key Features

  • Chrome's native i18n API (chrome.i18n.getMessage)
  • Standard _locales directory structure
  • HTML data-i18n attributes for automatic translation
  • JavaScript integration for dynamic content
  • Default locale: Turkish (as per original extension)
  • Full coverage: UI labels, buttons, messages, errors, and Jira descriptions
